#  Optimización de Riego Agrícola mediante Análisis de Datos

## Descripción del Proyecto
Este proyecto analiza la eficiencia de un nuevo sistema de riego IoT ("Ensayo") frente al método tradicional ("Regular") en un cultivo de flores de corte. Utilizando Python y estadística inferencial, se evaluó el impacto en las pérdidas por pudrición y la respuesta fisiológica de la planta.

## Objetivos
* Comparar tasas de pérdida entre sistemas de riego.
* Analizar la correlación entre variables climáticas (VPD) y temperatura de hoja.
* Validar estadísticamente la mejora operativa.

## Herramientas Utilizadas
* **Lenguaje:** Python 
* **Librerías:** Pandas, Seaborn, Matplotlib, Scipy.
* **Técnicas:** Test A/B, Prueba de Wilcoxon, ANOVA, Análisis de Correlación.

##  Resultados Clave
1. **Reducción de Pérdidas:** Se logró disminuir las mermas del **27% al 0.2%** en los lotes de prueba.
2. **Impacto Estadístico:** El tamaño del efecto (Cohen's d) fue de **0.91**, indicando una mejora drástica.
3. **Ahorro Económico:** Se proyecta un ahorro significativo por ciclo al reducir el desperdicio de planta.
